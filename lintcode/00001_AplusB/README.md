# 00001. A + B 问题

给出两个整数 a 和 b , 求他们的和并以整数（int）的形式返回。
 
## 题目大意
你不需要从标准输入流读入数据，只需要根据传入的两个参数 a 和 b，计算他们的和并返回就行。

## Example 1:
```
Input : a = 1, b = 2
Output : 3
```
## Example 2:
```
Input : a = -1, b = 1
Output : 0
```
